# OCW / Coursera-style Packaging Alignment

We treat OCW/Coursera as packaging and distribution patterns rather than “formats.”

Packaging goals:
- Exportable bundles: CurriculumPlan + UnitMap + referenced Mnemosyne assets.
- Deterministic manifests: content hashes + version pins.
- Omnichannel renderers: phone/chat/messaging/email + web.

Future exporters/importers:
- edX OLX-like emission (structure)
- IMS Common Cartridge (optional)
- Static site (OCW-style)
- “Course feed” JSON (Coursera-style metadata)
